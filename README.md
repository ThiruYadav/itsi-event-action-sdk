# ITSI Event Management SDK

## Installation

Install with `pip`:

```
virtualenv env
source env/bin/activate
pip install .
# Verify
python -c 'import itsi_event_management_sdk'
```

Run tests with:
```
python itsi_event_management_sdk/tests/tests.py
```
